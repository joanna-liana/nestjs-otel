# Median

A Medium clone using [this Prisma tutorial](https://www.prisma.io/blog/nestjs-prisma-rest-api-7D056s1BmOL0) as a starting point.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
